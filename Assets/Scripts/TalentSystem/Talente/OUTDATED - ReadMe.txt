Die hier liegenden Klassen sind aktuell nicht Konzept des Ability Systems, dessen Struktur sich noch im Aufbau befindet.

Mit so vielen Klassen für jede Spezialisierung wird das Wirrwarr zu groß, stattdessen wurde versucht ein Approach zu entwickeln,
in welcher eine einzelne Klasse (Die Ursprungsklasse) alle Informationen für jegliche Spezialisierung enthält. 

Das aktuelle Beispiel einer solche Klasse findet sich unter "Heal.cs"

Diese Skripte könnten jedoch als Template für Spells dienen, welche zukünftig wieder implementiert werden könnten.
